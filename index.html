The graph being drawn is queue length (y axis) against time (x axis).

In this example, I have used a base value of 2 calls per hour (distributed via poisson process), and a call response time of half an hour. The simulation stops after 24 simulation hours, and some extra results are posted to the console.

All of the variables for the simulation can be accessed and changed in the main.js source code.

Collecting Data

To collect data, I made an excel document with the results of the simulation (which I ran about 20 times).
